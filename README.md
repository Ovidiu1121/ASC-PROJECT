# Proiect ASM – Manipularea unui șir de octeți

## Descriere
Acest proiect este realizat în limbaj de asamblare pentru procesorul 8086,
în mod 16-biți (TASM/TLINK).

Programul citește între 8 și 16 octeți în format hexazecimal și realizează
operații pe biți, sortări și rotiri conform cerinței.

## Funcționalități
- Citirea octeților în format hex folosind INT 21h
- Calculul cuvântului C pe baza operațiilor bitwise
- Sortarea descrescătoare a șirului de octeți
- Determinarea poziției octetului cu cei mai mulți biți de 1
- Rotiri circulare și afișare în binar și hex
